## Fixes for TP-Link X80-5G
Personal repo for the X80-5G. 

Changes modem w_disable state pin to reflect the one used in original TP-Link DTS file and adds pcie support. 
Reasons for this can be found on [forum thread](https://forum.openwrt.org/t/issues-enabling-modem-on-tp-link-deco-x80-5g-not-working/250907)
